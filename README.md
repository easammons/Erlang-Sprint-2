# Project Title: Sudoku Solver 9000

This sprint project is used to introduce me to Erlang, so I decided to make a Sudoku solver, along with smaller uses such as a fibonacci maker. 

## Instructions for Build and Use

Steps to build and/or run the software:

1. Download VsCode 
2. Set up Erlang on your computer 
3. Download the code

Instructions for using the software:

1. Open up a terminal on VS code 
2. Start up erlang by typing in erl
3. Use Erlang commands according to the functions, such as sudoku:solve(Board).

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Vs code
* Erlang/OTP — version 26.x or later
* Error Lens
